# 📋 Sistema de Cadastro de Clientes

Sistema desenvolvido em Python para cadastro e gerenciamento de clientes, utilizando banco de dados SQLite para armazenamento das informações.

## ✨ Funcionalidades

- Cadastro de clientes
- Consulta de registros
- Edição de informações
- Exclusão de registros
- Visualização completa dos dados cadastrados
- Histórico de operações
- Registro de logs

```text
db/

```

## 🎯 Objetivos do Projeto

Este projeto foi desenvolvido para praticar:

- Operações CRUD (Create, Read, Update e Delete)
- Integração entre Python e banco de dados
- Desenvolvimento web com Django
- Manipulação de formulários
- Organização de templates
- Controle de dados utilizando SQLite

## 🚀 Como Executar

1. Clone o repositório

```bash
git clone https://github.com/nicole-m0/CRUD--cadastro-de-contatos
```

2. Instale as dependências

```bash
pip install django
```

3. Execute o projeto

```bash
python servidor.py
```
